# Portfolio Website

yokomachiのポートフォリオ用Webサイトです。
以下の要素を含んでいます。

- **Hero**: メインビジュアル表示
- **Profile**: 自己紹介と基本情報
- **Skill**: 技術スキルの一覧
- **Position**: ポジション歴
- **Certification**: 取得資格
- **Project**: これまでの主要プロジェクト
- **Link**: SNSリンク

## 技術スタック

Next.js, TypeScript, Tailwind CSS, shadcn/ui  
Hosting on Vercel
